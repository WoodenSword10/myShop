与下列代码比较：

always @(posedge clk or negedge rst_n)begin
  if (rst_n==1`b0)begin
    q<=0;
  end
  else begin
    q <= c;
  end
end
assign c = a+d;

从功能和综合的角度来说，两者完成一致。
第二个结构有利于仿真时观测a+d，也就是c的值，然后比对q和c的仿真波形，验证功能是否正确。
时序逻辑可以分成一个组合逻辑加一个D触发器
